# FUTURE_CS_02

## Phishing Detection & Awareness System

This project presents a professional phishing email analysis and awareness report based on 326 real-world email samples from public datasets.

## Dataset
- Enron Email Corpus (Legitimate Emails)
- Stanford Phishing Dataset
- Lehigh University Phishing Samples
- University of Washington Reported Emails

## Objective
- Identify phishing indicators
- Classify emails (Safe / Suspicious / Phishing)
- Explain attack techniques in simple language
- Provide awareness guidelines for users

## Tools Used
- Manual Email Analysis
- Email Header Analysis (Google Admin Toolbox)
- Dataset Analysis (CSV)
- Documentation (PDF)

## Repository Structure
- Report (PDF)
- Dataset (CSV)
- Sample Emails
- Supporting Evidence

## Disclaimer
All email samples are sourced from publicly available datasets used for educational and cybersecurity awareness purposes. No real user data is exposed.

## Outcome
This project demonstrates practical skills in phishing detection, risk analysis, and security awareness training aligned with real-world SOC analyst responsibilities.
